# Recipe Ace

This is a full-stack recipe management application called "Recipe Ace" using Next.js, React, TypeScript, ShadCN UI components, and Tailwind CSS. The application uses Genkit with the Gemini model for its core AI functionality.

## Core Features

- **AI-Powered Recipe Search**: Find recipes by name, cuisine, category, or even from specific restaurants.
- **Dynamic Cost Calculation**: Get location-specific ingredient pricing and calculate total recipe cost and cost per portion.
- **Interactive Editing**: View and edit recipe details, including ingredients, in an intuitive side-sheet form.
- **Infinite Scrolling**: Seamlessly load more recipes as you scroll.
